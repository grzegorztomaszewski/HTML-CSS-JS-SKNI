Układ elementów

  • { display: inline } - nie zaczyna nowej linii, nie ustala się wysokości i szerokości
  • { display: block }- rozpoczyna nową linię, można ustalać wielkość (domyślnie przyjmuje całą szerokość)
  • { display: inline-block }- jak inline, ale można ustalać wielkość
  • { display: none } - usuwa element

Układ elementów 2

  • { position: absolute } - relatywnie do niestatycznego przodka
  • { position: relative } - relatywnie do swojej normalnej pozycji
  • { position: fixed } - relatywnie do okna przeglądarki
  • { position: sticky} - relatywnie do scroll
